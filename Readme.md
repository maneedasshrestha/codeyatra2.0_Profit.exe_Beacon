# 🚀 CodeYatra 2.0

**CodeYatra 2.0** is an all-in-one community-driven platform specifically designed for college students. It bridges the gap between students by providing a centralized hub for academic resources, a specialized campus marketplace, and interactive community engagement.

---

## ✨ Key Features

### 📂 Academic Resource Vault
- **Community Curated**: Access a vast library of notes, PDFs, and study materials uploaded by fellow students.
- **Voting System**: A built-in upvote/downvote mechanism ensures the highest quality resources rise to the top.
- **Dynamic Previews**: View images and PDFs directly within the application for quick reference.

### 🛒 Student Marketplace
- **Peer-to-Peer Trading**: A dedicated space for students to buy, sell, or trade textbooks, lab equipment, and other college essentials.
- **Campus-Centric**: Focused specifically on your college community for safer and more convenient transactions.

### 💬 Community Feed
- **Stay Connected**: A real-time feed where students can share updates, ask questions, and stay informed about campus events.
- **Engagement**: Like, comment, and interact with posts to build a stronger campus network.

### 🤖 AI Student Advisor
- **Senior Persona**: An AI-powered assistant that acts as a knowledgeable "Senior Student" from your college.
- **Instant Guidance**: Get help with academic queries, project advice, or college-related questions powered by **Google Gemini**.

## 🛠️ Tech Stack

### Frontend
- **Framework**: [Next.js 16](https://nextjs.org/) (React 19)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

### Backend
- **Server**: [Express.js](https://expressjs.com/)
- **Database & Auth**: [Supabase](https://supabase.com/)
- **AI Engine**: [Google Gemini AI (1.5 Flash)](https://deepmind.google/technologies/gemini/)

---

## 🚀 Getting Started

1. **Clone the repository**
2. **Setup Frontend**:
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
3. **Setup Backend**:
   ```bash
   cd backend
   npm install
   # Configure your .env with Supabase and Gemini API keys
   npm run dev
   ```

## 🎯 Applicability
CodeYatra 2.0 is designed to solve the common challenges students face:
- **Information Fragment**: Consolidates scattered study materials into one accessible vault.
- **Economic Barrier**: Makes essential tools and books more affordable through a local student marketplace.
- **Social Connection**: fosters a digital community that translates into better on-campus relationships.
